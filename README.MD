# Melodify Music Streaming Web Application

A modern, responsive music streaming platform frontend built with HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript.

## Features

### 1. Landing Page
- Hero section with animated vinyl record
- Featured artists with hover effects
- How it works section with steps
- Pricing plans with highlighted popular option
- Responsive footer with newsletter signup

### 2. Admin Dashboard
- Sidebar navigation with multiple management sections
- Stats cards with growth indicators
- User management table with actions
- Chart placeholders for analytics
- Responsive layout with collapsible sidebar

### 3. Artist Dashboard
- Artist profile sidebar
- Upload music form (UI only)
- Song management table with streams and earnings
- Performance analytics placeholders
- Quick upload form and full upload modal

### 4. User/Fan Dashboard
- Music library with playable cards
- Recently played section
- Playlist management
- Recommendations with follow options
- Now playing bar with full player controls

## Technical Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom styles with CSS variables, animations, and responsive design
- **Bootstrap 5**: Grid system, components, and utilities
- **Vanilla JavaScript**: Interactive features without frameworks
- **Font Awesome**: Icons
- **Google Fonts**: Typography (Poppins, Montserrat)
